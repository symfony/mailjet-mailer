Mailjet Bridge
==============

Provides Mailjet integration for Symfony Mailer.

Configuration examples:

```dotenv
# API
MAILER_DSN=mailjet+api://$PUBLIC_KEY:$PRIVATE_KEY@default
# SMTP
MAILER_DSN=mailjet+smtp://$PUBLIC_KEY:$PRIVATE_KEY@default
```
